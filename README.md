## Installation ##
1. clone / download this repo
2. import `simple_tenantms.sql`
* login as: admin
* password: admin

if not import then:
1. copy and paste sql commands in `simple_tenantms.txt` & disregard table users
2. go to `localhost/simple-tms/pages/admin-signup.php`, sign up and login
